Chapter Zero is an online library. Online libraries act as a reservoir for the storage of important research data, information, and findings. Our system implements an online library system using Java and Spring Boot.
Our application is a 3 Tier MVC Web Application. The interactive GUI allows the User to login to the system and search for the desired resource from our Database. The search can be based on various keywords like the Title of the resource, Author of the resource or the publication. The user can choose the desired resource from the search results. The user can also view more about the author by the embedded link to Author’s profile among the search result. The user can favorite their resources and access them later.
The application will be leveraging Java features like:
• OOPS
• Exception Handling
• Database/JDBC
• Rest APIs
